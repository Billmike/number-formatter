Number Formatter
================

A small library that adds commmas to numbers

##  Installation

    npm install comma-bean

##  Usage

    var numberFormatter = require('comma-bean');
    var formattedNum = numberFormatter(35666)

  Output should be equal to 35,666

##  Test

    npm test
